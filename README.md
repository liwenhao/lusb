lusb
====

light weight usb wrapper for windows, easy way to get usb device path(hub port path)

## To Build

[Download FT Jam](http://freetype.org/jam/index.html#where-ftjam)

#### MS Visual C
```
> set JAM_TOOLSET=VISUALC
> jam
```

#### MINGW
`$ JAM_TOOLSET=MINGW jam`
